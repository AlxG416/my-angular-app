jsonplaceholder - плохое API, если обновить/удалить/создать пользователя, а после использовать метод GET, то изменения затруться. Я использовал его в качестве первоначальной подрузки данных пользователей, а после взаимодействовал с localStorage с помощью сервиса для последующей работы с пользователями.
