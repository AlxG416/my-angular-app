# MyAngularApp

Проект был сгенерирован с использованием [Angular CLI](https://github.com/angular/angular-cli) версии 21.2.11.

Деплой приложения доступен по ссылке: https://alxg416.github.io/my-angular-app/

При переходе по ссылке на деплой первичная отрисовка компонента с пользователями может не произойти в связи с блокировской запроса к серверу: необходимо попробовать подключиться через мобильный интернет, WiFi или включить VPN.

## Запуск приложения

Для запуска приложения необходимо сделать копию репозитория:

```bash
git clone https://github.com/AlxG416/my-angular-app.git
```

Перейдя в терминале в папку с проектом, выполнить пару команд:

```bash
npm install
npm start 
```

После выполнения команд, приложение будет доступно в браузере.
